# Regex Tester CLI

A Python command-line tool to test regular expressions.

## Description
This project allows users to enter a regex pattern and test it against text input.

## Features
- Regex matching
- Shows all matches
- Handles invalid patterns
- Useful for parsing and validation

## Technologies Used
- Python
- re module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python regex_tester.py

4. Enter regex pattern and text

## Example

Input:
\d+
Order 123 and 456

Output:
123
456
